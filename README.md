# Indonesia Tax Calculator

## Installation

```bash
git clone https://github.com/msofiullahs/taxcalculator.git
```

```nginx
cd ./taxcalculator
```

```composer
composer install
```

```nginx
cp .env.example .env
```

```artisan
php artisan key:generate
```

## Run PHPUnit Test

```artisan
php artisan test
```
