# Indonesia Tax Calculator

## Installation

```bash
git clone https://github.com/msofiullahs/taxcalculator.git
```

```bash
cd ./taxcalculator
```

```bash
composer install
```

```bash
cp .env.example .env
```

```bash
php artisan key:generate
```

## Run PHPUnit Test

```bash
php artisan test
```
